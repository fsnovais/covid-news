Corona News é um site criado com react para voce se manter informado sobre as novidades da covid-19


<h1>Rodando em localhost</h1>
Intalando as dependencias:<br />
<b>node install</b><br />
<b>yarn install</b><br />

Iniciando o servidor de aplicação<br />
<b>node start</b><br />
<b>yarn start</b><br />
