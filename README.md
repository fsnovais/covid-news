Corona News é um site criado com react para voce se manter informado sobre as novidades da covid-19


<h1>Rodando em localhost</h1>
Intalando as dependencias:
<b>node install</b>
<b>yarn install</b>

Iniciando o servidor de aplicação
<b>node start</b>
<b>yarn start</b>
